<h1> Unity FPS 게임 프로젝트</h1>

<p>
이 프로젝트는 Unity로 개발한 FPS(1인칭 슈팅) 게임입니다.<br>
1인칭과 3인칭 시점을 모두 지원하도록 구현되었습니다.
</p>

<hr>

<h2> 주요 기능</h2>
<ul>
    <li> 총알 발사 시스템</li>
    <li> 적 AI 및 체력 시스템</li>
    <li> 적 체력 관리</li>
    <li> 1인칭 / 3인칭 시점 전환</li>
    <li> 플레이어 이동 시스템</li>
    <li> 총 애니메이션</li>
    <li> 총알 충돌 및 애니메이션</li>
</ul>

<hr>

<h2> 사용 기술</h2>
<ul>
    <li>Unity Engine</li>
    <li>C#</li>
</ul>

<hr>

<h2> 조작 방법</h2>
<ul>
    <li>W / A / S / D - 이동</li>
    <li>마우스 - 시점 이동</li>
    <li>좌클릭 - 발사</li>
    <li>우클릭 - 조준</li>
    <li>키 (예: V) - 1인칭 / 3인칭 전환</li>
</ul>
